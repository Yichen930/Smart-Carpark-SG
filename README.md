# Smart Carpark Availability & Analytics System (Singapore)

## Quick Start

1. Open `index.html` or `search.html` in a web browser
2. No installation or build process required
3. All data is mocked - no backend needed

## Project Structure

- `index.html` - Entry point
- `search.html` - Search page
- `results.html` - Results page
- `detail.html` - Detail page
- `navigation.html` - Navigation page
- `analytics.html` - Analytics page
- `js/` - JavaScript files
